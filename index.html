HTML 8 Steps and Their CSS

Step HTML Structure CSS Class / Properties Purpose
1 Parent Container .header { width:100%; height:auto; } Main wrapper that covers full width
2 Component .componentSpacing { padding:4rem 0rem; } Breathing space between components
3 Component Container .screenSize { max-width:1200px; margin:auto; } Device screen size
4 Row .grid { display:grid; grid-template-columns:1fr 1fr; } Row layout with two columns
5 Column .dflex { display:flex; align-items:center; justify-content:center; } Flexible alignment inside column
6 Subcolumn .subColumn { width:100%; height:100%; } Nested columns take full space
7 Content Container h1 { font-family:'Nunito'; font-size:1.6rem; font-weight:500; } Text styling for content
8 Content Actual text, images, or media Final styled content shown to user

CSS and Bootstrap Classes Comparison

Step CSS Bootstrap Class Syntax
Parent Container .header { width:100%; height:auto; } w-100, h-auto <header class="w-100 h-auto"></header>
Component .componentSpacing { padding:4rem 0rem; } px-5 <section class="px-5"></section>
Component Container .screenSize { max-width:1200px; margin:auto; } container, container-fluid <div class="container"></div>
Row .grid { display:grid; grid-template-columns:1fr 1fr; } row <div class="row"></div>
Column .dflex { display:flex; align-items:center; justify-content:center; } col-12 col-sm-12 col-md-12 col-lg-12 <div class="col-12 col-sm-12 col-md-12 col-lg-12"></div>
Subcolumn .subColumn { width:100%; height:100%; } d-flex flex-column justify-content-center align-items-center w-100 h-100 <div class="d-flex flex-column justify-content-center align-items-center w-100 h-100"></div>
Content Container h1 { font-family:'Nunito'; font-size:1.6rem; font-weight:500; } fs-6, fw-bold, text-dark <p class="fs-6 fw-bold text-dark">Content</p>
Content Actual content (text/image) text-center, img-fluid, etc. <img src="img.jpg" class="img-fluid" alt="...">
